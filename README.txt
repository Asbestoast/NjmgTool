********************************************************************************
*                             Nekojara Monogatari                              *
*                        English Translation and Tools                         *
********************************************************************************

This contains an English translation and related tools for Nekojara Monogatari
for the Game Boy.

****************************************
*     Build Instructions (Windows)     *
****************************************

- Open the solution file in Visual Studio
- Set the startup project to "NjmgTool"
- Select "Build > Build Solution" from the menu
